Assessment 5: How can you remove white spacing between elements with display: inline-block ? Make the layout from below. Research and show various methods.

The space between inline elements is created because DOM treats them like alphabet words and creates an empty space " " after the inline-block elements. There are two methods to remove the space between inline-block elements.

inline
block

                <div class="inline-block">inline</div>
                <div class="inline-block">block</div>
            </div>

    inline
    block
