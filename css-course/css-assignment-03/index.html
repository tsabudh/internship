Assessment 3: Explore box-sizing property and how it helps to create an alternate box model. Show examples.

In the alternative box model, any width is the width of the visible box on the page. The content area width is that width minus the width for the padding and border.
To turn on the alternative model for an element, set box-sizing: border-box on it: 

            
            .box {
                    box-sizing: border-box;
                 }

This has box-sizing:border-box

This is some text within the div. The height of the border-box is 150px which sums as the height of contents with paddings and border thickness. ie: The declared height:150px defines the height including the border of the element in border-box.

This has box-sizing:content-box

This is some text within the div. The height of the content-box is also 150px but in content-box, the declared height:150px defines height of the content of the element. It does not include the border and padding of the element.
